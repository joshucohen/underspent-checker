Underspent Fund Checker

Quickly identify underutilized funds across multiple years

How it works

Upload an Excel file with 3 sheets (Year1, Year2, Year3). Each sheet should contain fund data for one year. The tool calculates average spending relative to available balance and flags funds below your selected threshold.

Analyze Your Data









Required format: 3 sheets named Year1, Year2, Year3
Columns in each sheet: fund_id, balance, spend, revenue

Definitions

  • Threshold (%): Maximum % of funds spent before flagged as underspent
  • Include Revenue: Adds incoming funds to available balance
  • Underspent: Spending rate below selected threshold
  • Requirements: Exactly 3 sheets (Year1, Year2, Year3) with consistent structure
